Project

Steam Review Sentiment Classification

Built a sentiment classifier on 50,000 Steam reviews using TF-IDF and compared Naive Bayes, KNN, and Decision Tree models.

PythonScikit-learnNLP
Overview
This project uses game review text to predict sentiment and compare how different simple machine learning models handle short opinion-heavy writing.
What I worked on
I prepared the review text, represented it with TF-IDF features, trained multiple classifiers, and compared their performance to understand the tradeoffs.
Built with
Python, Scikit-learn, NLP
