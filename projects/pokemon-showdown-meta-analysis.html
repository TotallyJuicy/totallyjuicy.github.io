Pokemon Showdown Meta Analysis

Overview
This project explores competitive Pokemon usage data to see which Pokemon, strategies, and trends shape the meta over time.

I parsed Smogon usage files, structured them into Pandas DataFrames, and looked for high-usage patterns and shifts that explain what players were bringing.
