Project

Spotify Listening Analytics Dashboard

Processed 8+ years of Spotify streaming history, removed passive listening, and built dashboard-ready datasets for Power BI.

PythonPandasPower BI
Overview
This project turns raw Spotify streaming history into cleaner listening data that can actually answer questions about habits, favorite artists, and changes over time.
What I worked on
I focused on cleaning the export, filtering out passive or noisy listening, shaping the data for dashboard use, and organizing it so Power BI could show patterns clearly.
Built with
Python, Pandas, Power BI
