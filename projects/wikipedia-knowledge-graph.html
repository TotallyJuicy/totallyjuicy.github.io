Project

Wikipedia Knowledge Graph

Built a linked-topic graph from Wikipedia pages with NetworkX and visualized centrality and clustering structure.

PythonNetworkXGephi
Overview
This project treats Wikipedia pages as a connected graph to explore how topics link together and which pages become important bridges.
What I worked on
I collected linked pages, built the graph structure with NetworkX, and used graph metrics and visualization tools to inspect centrality and clusters.
Built with
Python, NetworkX, Gephi
