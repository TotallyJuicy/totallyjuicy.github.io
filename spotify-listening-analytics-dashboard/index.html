Project

Spotify Listening Analytics Dashboard

Processed 8+ years of Spotify streaming history, removed passive listening, and built dashboard-ready datasets for Power BI.

Overview
This project takes raw Spotify listening history and turns it into cleaner data for understanding habits, artists, listening patterns, and changes over time.
What I worked on
I cleaned the export, filtered out passive listening, shaped the data with Python and Pandas, and prepared the final datasets for a Power BI dashboard.
